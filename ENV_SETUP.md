# Настройка переменных окружения

## Шаг 1: Создайте файл .env

Создайте файл `.env` в корне проекта со следующим содержимым:

```
VITE_OPENAI_API_KEY=ваш_api_ключ_здесь
```

## Шаг 2: Добавьте ваш API ключ OpenAI

Замените `ваш_api_ключ_здесь` на ваш реальный API ключ от OpenAI.

**Важно:** 
- Файл `.env` уже добавлен в `.gitignore` и не будет закоммичен в репозиторий
- Никогда не публикуйте ваш API ключ в открытом доступе
- Для продакшена используйте переменные окружения на сервере

## Шаг 3: Перезапустите dev сервер

После создания `.env` файла перезапустите dev сервер:

```bash
npm run dev
```

## Как это работает

При создании нового сценария приложение автоматически:
1. Анализирует товары в сценарии
2. Отправляет запрос к OpenAI API для генерации JTBD анализа
3. Показывает структурированный промпт для создания товарной страницы

Если API ключ не найден, приложение использует базовый шаблон JTBD.
