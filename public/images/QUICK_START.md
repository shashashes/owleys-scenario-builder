# Быстрый старт: добавление изображений

## Примеры товаров для добавления изображений:

### 1. Hanging Car Trunk Organizer "Hold&Go"
- **Item ID:** `p-3014-512`
- **SKU:** `OUTR02-05A`
- **Имя файла:** `p-3014-512.jpg` или `OUTR02-05A.jpg`

### 2. Dog Car Seat Cover "Travel Buddy"
- **Item ID:** `p-3014-20`
- **SKU:** `OUPS01-00A`
- **Имя файла:** `p-3014-20.jpg` или `OUPS01-00A.jpg`

---

## Шаги для добавления:

1. **Сохраните изображение** с именем по Item ID или SKU:
   - Например: `p-3014-20.jpg`, `p-3014-512.png`
   - Поддерживаемые форматы: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.svg`

2. **Поместите файл** в папку `public/images/`

3. **Запустите скрипт:**
   ```bash
   node scripts/add-images.js
   ```

4. **Готово!** Изображения появятся в приложении.

---

## Для всех товаров:

Используйте тот же процесс:
- Назовите файл по **Item ID** (например: `p-3014-20.jpg`)
- Или по **SKU** (например: `OUPS01-00A.jpg`)
- Поместите в `public/images/`
- Запустите `node scripts/add-images.js` — скрипт автоматически найдет все изображения и добавит их в CSV
